# nodejs-eml-renamer
Переименование eml по дате отправителю и получателю

Этот скрипт считывает файлы .eml из текущего каталога, извлекает информацию о дате и адресах электронной почты, очищает адреса электронной почты и переименовывает файлы на основе извлеченных данных.

Для запуска:
```
node index
```
